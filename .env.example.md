# `.env` 환경변수 가이드

이 문서는 프로젝트에서 사용할 `.env.local`, `.env.production`, 서버용 `.env` 파일에 어떤 값을 넣어야 하는지 정리한 예시이다.

## 1. 프론트엔드에서 사용하는 환경변수

브라우저에서 접근 가능한 값은 `NEXT_PUBLIC_` 접두사를 사용한다.

```env
NEXT_PUBLIC_APP_ENV=local
NEXT_PUBLIC_APP_NAME=민원나우
NEXT_PUBLIC_FRONTEND_BASE_URL=http://localhost:3000
NEXT_PUBLIC_API_BASE_URL=http://localhost:3000
NEXT_PUBLIC_KAKAO_MAP_APP_KEY=your_kakao_javascript_key
NEXT_PUBLIC_DEFAULT_LAT=37.5665
NEXT_PUBLIC_DEFAULT_LNG=126.9780
NEXT_PUBLIC_ENABLE_DEMO_MODE=true
```

### 설명
- `NEXT_PUBLIC_APP_ENV`
  - 실행 환경 구분값
  - 예: `local`, `development`, `production`
- `NEXT_PUBLIC_APP_NAME`
  - 화면에 표시할 서비스명
- `NEXT_PUBLIC_FRONTEND_BASE_URL`
  - 현재 프론트엔드 주소
  - 로컬 예: `http://localhost:3000`
  - 배포 예: `https://your-app.vercel.app`
- `NEXT_PUBLIC_API_BASE_URL`
  - 프론트가 호출할 백엔드 API 주소
  - 로컬 통합 실행이면 `http://localhost:3000`
  - Vercel + Lightsail 구조면 Lightsail API 도메인 사용
- `NEXT_PUBLIC_KAKAO_MAP_APP_KEY`
  - Kakao Map JavaScript API용 키
  - 브라우저에서 지도 렌더링에 사용
- `NEXT_PUBLIC_DEFAULT_LAT`
  - 기본 중심 좌표 위도
- `NEXT_PUBLIC_DEFAULT_LNG`
  - 기본 중심 좌표 경도
- `NEXT_PUBLIC_ENABLE_DEMO_MODE`
  - 심사용 demo 모드 노출 여부

## 2. 백엔드 서버에서 사용하는 환경변수

아래 값들은 브라우저에 노출되면 안 되므로 `NEXT_PUBLIC_` 없이 사용한다.

```env
NODE_ENV=development
PORT=3000
SERVER_BASE_URL=http://localhost:3000

PUBLIC_DATA_API_KEY=your_public_data_api_key
KAKAO_REST_API_KEY=your_kakao_rest_api_key
ODSAY_API_KEY=your_odsay_api_key

REQUEST_TIMEOUT_MS=10000
LOG_LEVEL=info

DEMO_MODE_DEFAULT=true
DEMO_DATA_PATH=./data/demo/recommendation-sample.json
```

### 설명
- `NODE_ENV`
  - Node 실행 환경
  - 예: `development`, `production`
- `PORT`
  - 백엔드 서버 포트
- `SERVER_BASE_URL`
  - 서버 자신의 기준 URL
- `PUBLIC_DATA_API_KEY`
  - 공공데이터포털 민원실 실시간 정보 API 키
- `KAKAO_REST_API_KEY`
  - Kakao Local REST API 키
  - 서버 전용 값이며 주소를 좌표로 바꾸는 단계에서 사용
  - 사용 모듈: [lib/server/kakao-local.ts](C:/dev/how_long_time/lib/server/kakao-local.ts)
  - 예시

```env
KAKAO_REST_API_KEY=your_kakao_rest_api_key
```

  - 사용 흐름 예시
    - 입력 주소 `서울시청`
    - Kakao Local REST API 호출
    - 결과 좌표 `lat=37.5666103`, `lng=126.9783882`
- `ODSAY_API_KEY`
  - ODsay API 키
  - 서버 전용 값이며 좌표 간 대중교통 이동시간 조회에 사용
  - 사용 모듈: [lib/server/odsay.ts](C:/dev/how_long_time/lib/server/odsay.ts)
  - 예시

```env
ODSAY_API_KEY=your_odsay_api_key
```

  - 사용 흐름 예시
    - 출발지 좌표와 민원실 좌표 입력
    - ODsay API 호출
    - 결과 `minutes`, `distanceKm` 형태의 이동시간 데이터 생성
- `REQUEST_TIMEOUT_MS`
  - 외부 API 호출 타임아웃
  - Kakao Local API, ODsay API 같은 외부 요청의 최대 대기 시간
  - 사용 모듈: [lib/server/env.ts](C:/dev/how_long_time/lib/server/env.ts),
    [lib/server/mobility.ts](C:/dev/how_long_time/lib/server/mobility.ts)
  - 예시

```env
REQUEST_TIMEOUT_MS=10000
```

  - 권장 기준
    - 로컬 개발 기본값: `10000`
    - 응답이 느린 환경에서 검토 가능한 값: `15000`
    - 너무 짧게 잡으면 fallback 처리 빈도가 높아질 수 있음
- `LOG_LEVEL`
  - 서버 로그 레벨
  - 예: `debug`, `info`, `warn`, `error`
- `DEMO_MODE_DEFAULT`
  - 심사용 시 기본 진입 모드를 demo로 둘지 여부
- `DEMO_DATA_PATH`
  - demo 모드에서 읽을 샘플 JSON 경로

## 3. Vercel 프론트 배포 시 필요한 핵심 변수

```env
NEXT_PUBLIC_APP_ENV=production
NEXT_PUBLIC_FRONTEND_BASE_URL=https://your-app.vercel.app
NEXT_PUBLIC_API_BASE_URL=https://api.your-domain.com
NEXT_PUBLIC_KAKAO_MAP_APP_KEY=your_kakao_javascript_key
NEXT_PUBLIC_ENABLE_DEMO_MODE=true
```

## 4. Lightsail 백엔드 배포 시 필요한 핵심 변수

```env
NODE_ENV=production
PORT=3000
SERVER_BASE_URL=https://api.your-domain.com

PUBLIC_DATA_API_KEY=your_public_data_api_key
KAKAO_REST_API_KEY=your_kakao_rest_api_key
ODSAY_API_KEY=your_odsay_api_key

REQUEST_TIMEOUT_MS=10000
LOG_LEVEL=info

DEMO_MODE_DEFAULT=true
DEMO_DATA_PATH=./data/demo/recommendation-sample.json
```

### mobility 연동용 최소 예시

```env
KAKAO_REST_API_KEY=your_kakao_rest_api_key
ODSAY_API_KEY=your_odsay_api_key
REQUEST_TIMEOUT_MS=10000
```

- 주소를 좌표로 변환하려면 `KAKAO_REST_API_KEY`가 필요하다.
- 좌표 간 대중교통 이동시간을 조회하려면 `ODSAY_API_KEY`가 필요하다.
- 외부 API 응답을 몇 초까지 기다릴지 정하는 값이 `REQUEST_TIMEOUT_MS`다.
- 현재 mobility 조합 계층은 [lib/server/mobility.ts](C:/dev/how_long_time/lib/server/mobility.ts)에서 관리한다.

## 5. 권장 파일 구성

- 프론트 로컬 개발: `.env.local`
- 백엔드 로컬 개발: `.env.local`
- Vercel 배포: Vercel 프로젝트 환경변수에 등록
- Lightsail 배포: 서버 내 `.env` 파일로 관리

## 6. 주의사항

- `PUBLIC_DATA_API_KEY`, `KAKAO_REST_API_KEY`, `ODSAY_API_KEY`는 절대 `NEXT_PUBLIC_`로 노출하지 않는다.
- Kakao 지도 렌더링용 JavaScript 키만 `NEXT_PUBLIC_KAKAO_MAP_APP_KEY`로 사용한다.
- 심사용 demo 모드를 쓸 경우 `DEMO_DATA_PATH`에 고정 데이터를 저장한다.
- 실제 키는 Git에 커밋하지 않는다.
